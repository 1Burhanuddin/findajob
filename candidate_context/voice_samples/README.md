# voice_samples/

Writing samples used by the cover letter writer role to match your voice and tone.

## What goes here

Plain text files (.txt) — typically:
- Cover letters you've written and sent
- Application essays you're proud of
- Writing samples that sound authentically like you

The more samples, the better calibration. 3–5 is a reasonable minimum.

## Naming

No required convention. Descriptive names help:

    amazon_2024_cover.txt
    meta_staff_eng_cover.txt
    personal_essay_2023.txt

## Gitignore

This directory is gitignored. **Do not commit writing samples.** They contain
personal voice, employer names, and context that should not be public.

Add your files here manually after cloning the repo.
