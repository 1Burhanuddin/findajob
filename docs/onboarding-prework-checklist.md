# Onboarding Pre-Work Checklist

Before you run the findajob onboarding interview, gather the items below.
Budget about 90 minutes for the interview itself — some people finish faster,
some take longer. You can pause mid-way as long as you return to the same
chat tab.

## Documents to have ready

- [ ] Up-to-date resume (PDF or Word doc).
- [ ] Any performance reviews, 360 feedback, or self-assessments you'd like
      the interview to read. Optional but valuable — the more context the
      model has, the fewer gap-fill questions it needs to ask you.
- [ ] Prior cover letters (optional).
- [ ] Your LinkedIn profile URL.

## Answers to have ready

- [ ] One sentence describing the role you want next.
- [ ] A rough list of categories of jobs you never want to see, even if the
      title technically matches your target (e.g. "no substitute teaching",
      "no child protective services", "no sales engineering").
- [ ] 5–15 employers you're most interested in working for.

## Pick your LLM

- [ ] Pick one: Claude (claude.ai) / ChatGPT (chatgpt.com) / Gemini
      (gemini.google.com).
- [ ] Confirm you're on a paid reasoning-grade tier:
      - Claude: Pro ($20/mo) — select **Claude Opus 4.7** and enable
        **Extended thinking**.
      - ChatGPT: Plus ($20/mo) — select **GPT-5.4 Thinking** from the
        model picker.
      - Gemini: Google AI Pro ($19.99/mo) — select **Gemini 3.1 Pro**.
- [ ] Open a new chat in your chosen platform.

## Running the interview

- [ ] In your stack's web UI, visit `/onboarding/`. Click "Copy the
      interview prompt" and paste it into the new chat you opened.
- [ ] Follow the phase prompts. Upload your documents when asked.
- [ ] When the LLM has emitted all seven file blocks, copy the entire
      chat (or just the seven blocks — the parser handles either) and
      paste it into the text box at the bottom of `/onboarding/`.
      Click "Inject config". The pipeline is configured.
