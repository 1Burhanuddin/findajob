Set up Gmail job-alert ingestion

If you've enabled job-alert emails (LinkedIn, Indeed, or any other source) and want findajob to ingest them automatically, connect your Gmail account here. This step is optional — every other ingestion path (career-page feeds, the search APIs you've already configured) keeps working without it.

Prerequisites — 2-Factor Authentication on the Google account

Google requires 2FA to be enabled on your account before app passwords can be generated. If your account isn't yet on 2FA, enable it at myaccount.google.com/security first, then return to this page.

Once 2FA is on, generate a new app password labeled "findajob" at myaccount.google.com/apppasswords and paste it into the form below.

{% include 'gmail_config/_card.html' %}

When you're done: click Test connection above. Once the status pill shows Authorized, click Continue to dashboard below. If you'd rather skip Gmail ingestion entirely (or come back to this later), use Skip for now — you can re-open this step any time via /onboarding/?mode=rerun or directly at /config/gmail/.
